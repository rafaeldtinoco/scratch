Add azure-cli to all nodes by adding:

	deb [arch=amd64] https://packages.microsoft.com/repos/azure-cli/ bionic main

to /etc/apt/sources.list and then executing:

	$ sudo apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys EB3E94ADBE1229CF

before updating the package list with:

	$ sudo apt-get update -y

and installing azure-cli package with:

	$ sudo apt-get install -y azure-cli

After having azure-cli properly installed, execute "az login" in all cluster nodes:

	$ az login
